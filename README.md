# 김비서 : 사용자 맞춤 취업 AI 비서
Team.KHPT

## 실행하기
`manage.py`와 같은 위치에 아래와 같이 `.env` 파일을 추가해주세요.
```
OPENAI_KEY=<openai-api-key>
SARAMIN_KEY=<saramin-api-key>
DEBUG=True
SECRET_KEY=1234
```
모듈 설치
```
$ pip install -r requirements.txt
```
서버 실행
```
$ python manage.py runserver
```
